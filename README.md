# node-express-mongo-todos-app
- `npm init -y`
- `npm install express mongoose`
- `npm i -D nodemon`

## Change in package-json: `"scripts": {
    "start": "node index",
    "dev": "nodemon index"
  }`
## Create index.js in main folder
## To run app:
- `npm run dev`
## Register mongodb (data base): "https://www.mongodb.com/"
## NPM HBS:
- `npm i express-handlebars`
